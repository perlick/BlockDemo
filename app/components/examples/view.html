Basic Usage

To move a block on the board, type an instruction in the text area and click submit. In this example, the instruction requests that the far left block be moved to the top right. You can see that the specified block is moved to the correct location.

Reset & Shuffle Buttons

The Shuffle button randomizes all block locations. This can be used to quickly test an instruction on different board configurations.

In contrast to Shuffle, the Reset button can be used to test different instructions on the same board. The Reset button will undo all instructions since the last Shuffle or since the page laod
